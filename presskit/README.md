# Lentando

**Progress At Your Pace** üê¢ A habit and substance use tracker that runs entirely in your browser.

## Key Features

- Offline, local-first tracking with optional sync
- One-tap logging for use, resists, and healthy actions
- Visual trends + editable day-by-day history
- 50+ badges for gaps, streaks, milestones, mindful choices
- Installable PWA across phone, tablet, desktop
- Open source under GPLv3, no ads, no data selling
